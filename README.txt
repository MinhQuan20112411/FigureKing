# Thực tập tốt nghiệp Web bán mô hình đồ chơi

FigureKing

# Cách mở dự án

cài đặt xampp, nodejs

trong xampp start Apache và MySQL

import file FigureKing.sql lên phpMyAdmin.net

mở thư mục SOURCE CODE trong Visual Studio Code

Mở trang chủ
trong terminal ở đường dẫn SOURCE CODE\FigureKing\backend chạy npm i và node server.js 
trong terminal ở đường dẫn SOURCE CODE\FigureKing\frontend chạy npm i và npm start

Mở trang admin
trong terminal ở đường dẫn SOURCE CODE\Admin-main\Admin-main\backend chạy npm i và node server.js 
trong terminal ở đường dẫn SOURCE CODE\Admin-main\Admin-main\frontend chạy npm i và npm start
